Week 12
NoSQL

This week, I learnt the benefits and uses of NoSQL databases. NoSQL, also known as ‚ÄúNot only SQL,‚Äù is a type of database designed for storing and retrieving unstructured data. It can manage a vast amount of data and features a dynamic schema. Unlike traditional databases, NoSQL doesn‚Äôt have a specific query language or many relationships. Instead, it organizes data into collections and documents, with each database potentially containing numerous collections, and each collection housing multiple documents.

Some benefits NoSQL databases offer over SQL databases include better suited handling of unstructured data and large datasets. They can manage high write loads, provide high availability in unstable environments, and handle location-based data. These features make NoSQL a versatile choice for many data management needs.

Bootstrap

This week, I have learnt about the uses of Bootstrap, especially when creating User Interfaces and resolving issues imposed by various browsers and devices. Its Flexbox grids and device breakpoints make it easier and faster for developers to design and create a website. Using CSS libraries like Bootstrap has made it so that I do not have to "memorize stuff", but instead focus on creating "new stuff".
